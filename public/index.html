An iOS developer I worked with had to generate random product names for an app we were working on. He was called Avario (https://github.com/avario) so when I ported it from Swift to Javascript, I decided to name it after him. I hooked it up to a recipe API so you can make Avario's amazing creations, like "Quite Boiled Chicken" or "Really Flaky Tuna Pie". Enjoy. You can read the (terrible, messy) source code here
